About ZARS

ZARS is an online reservation application. Usage of ZARS is subject to invitation only. To use the reservation application you must have been invited by the site operator and a user account must have been created for you. It is not possible to sign up for a user account.

Supported Internet Browsers and Settings

ZARS has been developed and tested with Mozilla Firefox 3.6 on Windows but should work with a recent version of all major Internet Browsers. Javascript must be enabled in order to use ZARS.
