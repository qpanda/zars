Hilfe

ZARS ist eine online Reservierungsanwendung. Die Verwendung von ZARS ist ausschließlich auf Einladung hin möglich. Um die Reservierungsanwendung nutzen zu können müssen Sie vom Seitenbetreiber eingeladen werden und ein Benutzerkonto muss für Sie angelegt worden sein. Es ist nicht möglich selbst ein Benutzerkonto anzulegen.

Unterstützte Internet Browsers and Einstellungen

ZARS wurde mit Mozilla Firefox 3.6 unter Windows entwickelt und getestet sollte aber mit der aktuellen Versionen aller gängiger Internet Browser funktionieren. Javascript muss aktiviert sein um ZARS verwenden zu können.
